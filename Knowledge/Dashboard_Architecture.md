> **PDD-00 P4 输出 · 永久文档（Product SSoT）**
> 更新时间：2026-07-06
> **真正经营驾驶舱。全部采用统一 Summary Card。**

---

## 一、Dashboard 定位

Dashboard 是 FinanceDesk 的**唯一经营数据总览页面**。登录后默认进入。

| 属性 | 值 |
|:----:|-----|
| 角色 | 经营驾驶舱 |
| 数据来源 | Summary Engine（非直接 ORM） |
| 权限 | 全部角色可访问 |
| 刷新 | 每次进入重新计算 |
| 布局 | Summary Card Row + Chart + Alert List |

## 三、Summary Card 规范

| 卡片 | 数据来源 | 计算方式 |
|:-----|:---------|:---------|
| **收入** | IncomeFlow | 当月 `sum(taxable_amount)` |
| **成本** | CostFlow | 当月 `sum(taxable_amount)` |
| **利润** | 收入 - 成本 | 实时计算 |
| **执行率** | 已执行 / 合同额 | Summary Engine |
| **回款率** | 已收款 / 应收款 | Summary Engine |
| **Gap** | 合同 - 实绩 | Summary Engine |

---

## 四、经营趋势图

| 属性 | 值 |
|:-----|:-----|
| 类型 | 柱状图 + 折线图组合 |
| 维度 | 近 12 个月 |
| 数据 | 收入（柱）+ 成本（柱）+ 利润（线） |
| 交互 | 悬停显示数值 |

---

## 五、经营预警

| 预警类型 | 条件 | 颜色 |
|:---------|:-----|:-----|
| 成本超额 | 实际成本 > 预算 110% | 🔴 红 |
| 回款逾期 | 已开票超 90 天未回款 | 🟠 橙 |
| Gap 差距 | Gap > 合同额 20% | 🔴 红 |
| 导入失败 | 最近导入失败数 > 0 | 🟠 橙 |
| Summary 异常 | Revenue ≠ Cost 镜像断开 | 🔴 红 |

---

## 六、Dashboard 数据流

```
Dashboard Page
    │
    ├── SummaryCard（收入/成本/利润）→ Summary Engine → Repository
    ├── 趋势图 → Summary Engine（按月聚合）
    ├── 预警 → Rule Engine（条件匹配）
    └── 最近导入 → ImportBatch Repository
```
