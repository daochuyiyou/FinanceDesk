# ERP Import Sandbox — 导入沙箱

> **BDD-06E Phase 1 输出 · 永久文档（SSoT）**
> 更新时间：2026-07-05
> **Sandbox：仅预览，不写入业务表。所有写入业务表的操作走 Phase 3 正式导入。**

---

## 一、定位

ERP Import Sandbox 是 ERP 数据从文件上传到正式导入业务表之前的**预览验证环境**。

| 属性 | 值 |
|:----:|-----|
| 角色 | 非破坏性预览层 |
| 输入 | Excel 文件 |
| 输出 | Preview 报告（不持久化） |
| 可写 | ❌ 不写入任何业务表 |
| 可删 | 预览数据不持久化 |

---

## 二、预览流程

```
Excel 上传
    ↓
① ERP Fact Preview  ← 解析结果预览（列映射 / 行数 / 错误）
    ↓
② Match Preview     ← 自动匹配预览（P1~P4 匹配结果）
    ↓
③ Import Preview    ← 导入摘要（分类统计）
    ↓
[确认导入] → Phase 3 正式导入
```

### 流程规则

| 规则 | 说明 |
|:----:|------|
| **预览不持久化** | 所有预览数据仅在内存中处理，不写入 `erp_staging_flow` |
| **预览可重复** | 同一文件多次上传预览结果一致 |
| **预览可放弃** | 用户可随时放弃，不留下任何数据痕迹 |
| **确认后才入库** | 仅在用户点击"确认导入"后写入 `erp_staging_flow` |

## 三、ERP Fact Preview

### 输入

| 字段 | 类型 | 说明 |
|:----:|:----:|------|
| file | .xlsx / .xls | 待解析的 ERP Excel 文件 |

### 输出

| 字段 | 类型 | 说明 |
|:----:|:----:|------|
| sheet_name | String | 工作表名称 |
| parsed_rows | Int | 解析后的有效行数 |
| skipped_rows | Int | 跳过的无效行（空行/合计行） |
| errors | List[String] | 解析告警信息 |
| columns_mapped | Dict | 列映射关系（中→英） |
| sample_data | List[Dict] | 前 5 条示例数据 |

### 列映射

| 英文字段 | 中文匹配 | 说明 |
|:--------:|---------|------|
| occur_date | 发生时间/单据日期 | 发生日期 |
| erp_record_id | 凭证号/业务单号 | 唯一标识 |
| summary | 摘要/说明 | 交易说明 |
| amount_in | 贷方金额/收入 | 收入金额 |
| amount_out | 借方金额/支出 | 支出金额 |
| raw_project_name | 客户名称/项目名称 | 项目/业主名 |

### 预览 UI


## 四、Match Preview

### 匹配引擎（引用 BDD-06D）

| 优先级 | 匹配方式 | 说明 |
|:------:|---------|------|
| P1 | `order_no` 精确匹配 | voucher_no == order_no |
| P2 | `contract_no + amount` | 按合同+金额组合 |
| P3 | `owner_name + business_date` | 按业主+日期模糊 |
| P4 | 人工确认 | 人工选择 |

### 匹配结果

| 状态 | 含义 | 数量 |
|:----:|------|:----:|
| 🟢 AUTO_MATCH | 自动匹配成功 | N |
| 🟠 MULTI_MATCH | 多候选 | N |
| 🔴 UNMATCHED | 未匹配 | N |

### 预览 UI

```
┌─ Match Preview ─────────────────────────────────────┐
│  P1(order_no):     85 条 ✅                          │
│  P2(contract+amt): 30 条 ✅                          │
│  P3(name+date):    15 条 ⚠️ 低置信度               │
│  P4(人工):         10 条 ❌                          │
│ ─────────────────────────────────────────           │
│  自动匹配: 130 条 │ 人工待定: 10 条 │ 异常: 0 条     │
│ [确认] → Import Preview  │  [返回]                  │
└──────────────────────────────────────────────────────┘
```

| 统计项 | 说明 |

## 六、API 接口（Sandbox 模式）

| 方法 | 路径 | 说明 |
|:----:|------|------|
| POST | `/api/v1/erp/sandbox/upload-preview` | 上传并预览，不持久化 |
| POST | `/api/v1/erp/sandbox/match-preview` | 模拟匹配，返回匹配结果 |
| GET  | `/api/v1/erp/sandbox/import-preview` | 当前暂存数据的摘要统计 |

---

## 七、约束

| 规则 | 说明 |
|:----:|------|
| **Sandbox 不持久化** | 所有预览操作不写入数据库 |
| **Sandbox 不可逆** | 预览不可回退，但可重复上传 |
| **Sandbox 不锁定** | 多人可同时预览不同文件 |
| **Sandbox 不覆盖** | 预览结果不覆盖已存在的素材数据 |
