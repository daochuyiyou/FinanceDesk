## 一、完整菜单树

基于 BDD-00 确认的业务域划分（经营业务域 / 基础资料域 / ERP 集成 / 经营驾驶舱），重新组织菜单。

```
FinanceDesk
│
├── 📋 经营管理
│   ├── ① 合同中心            ← 框架/单项合同（含业主信息）
│   ├── ② 订单中心            ← 唯一经营结算单元
│   ├── ③ 收入管理            ← 开票记录
│   ├── ④ 成本执行            ← 成本支出（引用合同库）
│   ├── ⑤ 收款管理            ← 回款跟踪
│   └── ⑥ 付款管理            ← 付款跟踪
│
├── 🔄 财务集成
│   └── ⑦ 财务集成核对        ← ERP 同步 + 差异对账
│
├── 📊 经营分析
│   └── ⑧ 经营驾驶舱          ← Dashboard（只读分析层）
│
├── 📁 基础资料
│   ├── ⑨ 成本供应商合同库     ← 年度合同 + 单价体系
│   ├── ⑩ 系统字典             ← 枚举值管理
│   ├── ⑪ Excel 模板管理       ← 导入导出模板
│   └── ⑫ 参数配置             ← 系统参数

--- 非菜单页面（导航跳转进入）---
│   ├── ⑬ 订单明细             ← 从订单中心/合同中心跳转
│   ├── ⑭ 项目明细             ← 从合同中心跳转
│   ├── ⑮ 数据导入入口         ← 各模块内的导入功能
│   └── ⑯ 操作日志             ← 审计日志（右下角/设置入口）
```

### 菜单分组

| 分组 | 包含页面 | 权限 |
|------|---------|------|
| **经营管理** | 合同中心、订单中心、收入管理、成本执行、收款管理、付款管理 | 全部用户 |
| **财务集成** | 财务集成核对 | 财务人员 |
| **经营分析** | 经营驾驶舱 | 管理层 |
| **基础资料** | 成本供应商合同库、系统字典、Excel 模板管理、参数配置 | 管理员 |

## 二、页面职责

### ① 合同中心（对应：ProjectList / ProjectDetail）

| 属性 | 值 |
|------|-----|
| 页面作用 | 管理框架合同和单项合同，包含甲方业主信息 |
| 允许维护的数据 | 合同名称、签订日期、有效期、项目类型、集团内外、业主信息 |
| 禁止维护的数据 | 订单数据、流水数据、回款/支付数据、成本数据 |
| 对应业务对象 | Project |
| 数据来源 | 人工录入 / Excel 导入 / ERP 同步（erp_no 字段） |
| 是否允许导入 | ✅ |
| 是否允许导出 | ✅（BDD 阶段补齐） |
| 是否允许修改 | ✅（仅限于非 ERP 来源字段） |
| 是否允许删除 | ✅（逻辑删除） |
| 子页面 | ProjectDetail（查看项目详情） |

**职责边界**：合同→输出订单中心。合同中心不管理订单、不录流水。

---

### ② 订单中心（对应：OrderPage / OrderDetail / OrderManagement）

| 属性 | 值 |
|------|-----|
| 页面作用 | 管理订单信息 —— 唯一经营结算单元 |
| 允许维护的数据 | 订单编号、订单名称、金额（含税/不含税）、订单类型、状态、甲方单位 |
| 禁止维护的数据 | 收入流水、成本流水、回款/支付 |
| 对应业务对象 | Order |
| 数据来源 | 人工录入 / Excel 导入 / ERP 同步 |
| 是否允许导入 | ✅ |
| 是否允许导出 | ✅（BDD 阶段补齐） |
| 是否允许修改 | ✅ |
| 是否允许删除 | ✅（逻辑删除，需检查关联流水） |
| 子页面 | OrderDetail（查看订单明细 + 关联的流水/收付款） |

**职责边界**：订单中心承接合同中心，衍生收入管理和成本执行。
**引用**：合同中心（project_id）、成本供应商合同库（supplier_id）

---

### ③ 收入管理（对应：IncomeManagement）

| 属性 | 值 |
|------|-----|
| 页面作用 | 按订单维度记录开票收入 |
| 允许维护的数据 | 发票号码、开票金额（含税/不含税）、税率、开票日期、状态 |
| 禁止维护的数据 | 订单本身、回款(由收款管理维护)、成本数据 |
| 对应业务对象 | IncomeFlow |
| 数据来源 | 人工录入 / Excel 导入 / ERP 同步 |
| 是否允许导入 | ✅ |
| 是否允许导出 | ✅（BDD 阶段补齐） |
| 是否允许修改 | ✅ |
| 是否允许删除 | ✅（逻辑删除，需检查关联回款） |

**职责边界**：按 order_id 归属，不创建/修改订单。输出→收款管理。

---

### ④ 成本执行（对应：CostExecution）

| 属性 | 值 |
|------|-----|
| 页面作用 | 按订单维度记录成本支出 |
| 允许维护的数据 | 成本方、成本类型、成本金额（含税/不含税）、税率、科目、对应预算项 |
| 禁止维护的数据 | 供应商合同信息（由合同库维护）、付款（由付款管理维护） |
| 对应业务对象 | CostFlow |
| 数据来源 | 人工录入 / Excel 导入 / ERP 同步 |
| 是否允许导入 | ✅ |
| 是否允许导出 | ✅（BDD 阶段补齐） |
| 是否允许修改 | ✅ |
| 是否允许删除 | ✅（逻辑删除，需检查关联付款） |

**职责边界**：引用成本供应商合同库（supplier_id），不修改合同信息。输出→付款管理。

---

### ⑤ 收款管理（对应：CollectionManagement / CollectionPage）

| 属性 | 值 |
|------|-----|
| 页面作用 | 记录实际回款，按收入流水追溯 |
| 允许维护的数据 | 回款日期、回款金额、收款凭证号 |
| 禁止维护的数据 | 收入流水本身、订单信息 |
| 对应业务对象 | Collection |
| 数据来源 | 人工录入 / Excel 导入 |
| 是否允许导入 | ✅ |
| 是否允许导出 | ✅（BDD 阶段补齐） |
| 是否允许修改 | ✅ |
| 是否允许删除 | ✅（逻辑删除） |

**职责边界**：引用收入流水（flow_id），不创建/修改收入流水。

---

### ⑥ 付款管理（对应：PaymentManagement）

| 属性 | 值 |
|------|-----|
| 页面作用 | 记录实际付款，按成本流水追溯 |
| 允许维护的数据 | 支付对象、支付日期、支付金额、凭证号 |
| 禁止维护的数据 | 成本流水本身、供应商合同信息 |
| 对应业务对象 | Payment |
| 数据来源 | 人工录入 / Excel 导入 |
| 是否允许导入 | ✅ |
| 是否允许导出 | ✅（BDD 阶段补齐） |
| 是否允许修改 | ✅ |
| 是否允许删除 | ✅（逻辑删除） |

**职责边界**：引用成本流水（cost_id），不创建/修改成本流水。

---

### ⑦ 财务集成核对（对应：DataHub）

| 属性 | 值 |
|------|-----|
| 页面作用 | ERP 数据同步、差异核对、匹配管理 |
| 允许维护的数据 | 匹配规则、差异标记 |
| 禁止维护的数据 | ERP 来源的业务数据（项目/订单/流水等） |
| 对应业务对象 | ERP 集成 |
| 数据来源 | ERP 导入 |
| 是否允许导入 | ✅（仅限 ERP 同步） |
| 是否允许导出 | ✅ |
| 是否允许修改 | ❌（不可修改 ERP 数据） |
| 是否允许删除 | ❌ |

**职责边界**：不修改 ERP 数据。只能导入、匹配、分析。

---

### ⑧ 经营驾驶舱（对应：Dashboard）

| 属性 | 值 |
|------|-----|
| 页面作用 | 经营分析看板，只读聚合展示 |
| 允许展示的数据 | 项目汇总、订单利润、应收账龄、Gap、回款率、成本率 |
| 禁止维护的数据 | **全部数据均不可维护** |
| 对应业务对象 | Dashboard（聚合查询） |
| 数据来源 | 系统自动计算（R006） |
| 是否允许导入 | ❌ |
| 是否允许导出 | ✅（报表导出） |
| 是否允许修改 | ❌ |
| 是否允许删除 | ❌ |

**禁止行为（R005 + F004）**：表单输入、提交按钮、状态切换、数据录入。

---

### ⑨ 成本供应商合同库（对应：SupplierPage + SupplierContractManagement + SupplierUnitPriceManagement）

| 属性 | 值 |
|------|-----|
| 页面作用 | 维护年度合同单价体系（Master Data） |
| 允许维护的数据 | 合同编号、合同名称、年度、有效期、普工单价、技工单价、综合单价 |
| 禁止维护的数据 | 成本数据、付款数据、订单信息 |
| 对应业务对象 | Supplier, SupplierContract, SupplierUnitPrice |
| 数据来源 | 人工录入 / Excel 导入 |
| 是否允许导入 | ✅（合同模板 + 单价模板） |
| 是否允许导出 | ✅（BDD 阶段补齐） |
| 是否允许修改 | ✅ |
| 是否允许删除 | ✅（逻辑删除，不影响已引用的成本记录） |

**禁止行为（R008 + F007）**：不得录成本、不得录付款、不得参与经营流程。
**重命名**：当前代码中称为"供应商管理""合同管理""单价管理"，BDD 阶段应统一更名为"成本供应商合同库"。

---

### ⑩ 系统字典（当前无独立页面）

| 属性 | 值 |
|------|-----|
| 页面作用 | 管理系统中所有枚举值（项目类型、订单状态、成本类型等） |

| 禁止维护的数据 | 业务数据 |
| 对应业务对象 | SysDictionary |
| 数据来源 | 人工维护 |
| 是否允许导入 | ✅ |
| 是否允许导出 | ✅ |
| 是否允许修改 | ✅ |
| 是否允许删除 | ✅（需检查引用关系） |

**职责边界**：仅维护枚举值，不存储业务记录。
**当前状态**：仅有 API 支持（`/dict/{category}`），无独立前端页面。BDD 阶段需创建。

## 三、页面职责矩阵（P2）

### 维护 → 引用 → 输出

| 页面 | 维护对象 | 引用对象 | 输出对象 |
|------|---------|---------|---------|
| ① 合同中心 | 合同（含业主信息） | — | 订单中心 |
| ② 订单中心 | 订单 | 合同中心、成本供应商合同库 | 收入管理、成本执行 |
| ③ 收入管理 | 收入流水（开票） | 订单中心 | 收款管理 |
| ④ 成本执行 | 成本流水 | 订单中心、成本供应商合同库 | 付款管理 |
| ⑤ 收款管理 | 回款 | 收入管理 | 经营驾驶舱 |
| ⑥ 付款管理 | 付款 | 成本执行 | 经营驾驶舱 |
| ⑦ 财务集成核对 | 匹配规则/差异标记 | ERP 数据 | 经营驾驶舱 |
| ⑧ 经营驾驶舱 | **无** | 全部业务模块 | 报表/分析 |
| ⑨ 成本供应商合同库 | 年度合同+单价 | — | 成本执行 |
| ⑩ 系统字典 | 枚举值 | — | 全部模块 |
| ⑪ Excel 模板管理 | 模板文件 | — | 全部模块 |
| ⑫ 参数配置 | 系统参数 | — | 系统行为 |
