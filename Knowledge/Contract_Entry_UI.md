## 三、合同类型选择交互流程

```mermaid
stateDiagram-v2
    [*] --> 选择合同类型：用户打开新增页

    选择合同类型 --> 框架合同模式：点击"框架合同"
    选择合同类型 --> 单项合同模式：点击"单项合同"

    框架合同模式 --> 框架合同_填写：显示"所属框架"字段
    框架合同_填写 --> 选择已有框架：点击下拉
    框架合同_填写 --> 新建框架：点击"新建框架"按钮
    选择已有框架 --> 填充框架信息：自动填充年度/有效期
    新建框架 --> 弹出框架新建表单

    单项合同模式 --> 单项合同_填写：隐藏"所属框架"字段
    单项合同_填写 --> 直接填写：无框架关联

    框架合同_填写 --> 保存前校验
    单项合同_填写 --> 保存前校验

    保存前校验 --> 保存成功：校验通过
    保存前校验 --> 校验失败：字段错误

    保存成功 --> [*]：跳转到详情页
    校验失败 --> 对应模式：保留已填数据
```

## 五、响应式布局说明

| 断点 | 布局 | 说明 |
|:----:|:----:|------|
| ≥ 1200px | 双列 | 字段左右布局，左侧必填，右侧可选 |
| 768-1199px | 单列 | 字段上下排列 |
| < 768px | 单列 + 隐藏折叠 | 折叠区域默认收起 |
