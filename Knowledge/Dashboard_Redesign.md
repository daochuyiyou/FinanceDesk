## 二、各区域说明

### Business Analyzer

| 选择器 | 类型 | 选项 |
|:-------|:-----|:------|
| 经营期间 | 下拉 | 已导入的月度列表 |
| 分析视角 | Radio | 公司 / 合同 / 项目 / 订单 |
| 分析对象 | 下拉 | 根据视角动态 |
| 负责人 | 下拉 | 全部 / 具体负责人 |
| 合同类型 | 下拉 | 全部 / 框架合同 / 单项合同 |

### KPI Summary

| 指标 | 计算方式 | 说明 |
|:-----|:---------|:------|
| 合同金额 | 当前期间合同总额 | — |
| 收入 | 当前期间收入合计 | IncomeFlow |
| 成本 | 当前期间成本合计 | CostFlow |
| 利润 | 收入 - 成本 | 自动推导 |
| 回款率 | 已收款 / 应收款 | Collection / IncomeFlow |
| 付款率 | 已付款 / 应付款 | Payment / CostFlow |
| 收入 Gap | 合同额 - 收入 | 差异越大越异常 |
| 成本 Gap | 合同额 - 成本 | 差异越大越异常 |

### Monthly Trend

| 属性 | 值 |
|:-----|:-----|
| 时间范围 | 近 12 个月 |
| 维度 | 收入（线）+ 成本（线）+ 利润（柱） |
| 粒度 | 按月 |
| 禁止 | ❌ 今日、昨日、最近 7 天 |

### Business Alerts

| 预警 | 触发条件 | 点击动作 |
|:-----|:---------|:---------|
| ERP 未匹配 | 存在 `match_status=pending` 流水 | → ERP Workbench |
| 收入 Gap | 合同额 - 收入 > 合同额 20% | → 收入明细 |
| 成本 Gap | 合同额 - 成本 > 合同额 20% | → 成本明细 |
| 超期回款 | 开票超 90 天未回款 | → 收款管理 |
| 超期付款 | 成本超 90 天未付款 | → 付款管理 |

### Business Actions

| 事项 | 数据来源 | 点击动作 |
|:-----|:---------|:---------|
| 待人工匹配 | `match_status=pending` | → Matching Workbench |
| 待确认收入 | IncomeFlow.status="待回款" | → 收入管理 |
| 待确认成本 | CostFlow.status="待支付" | → 成本管理 |
| 待确认付款 | Payment.status="待支付" | → 付款管理 |
| 待处理项目 | Project 状态异常 | → 项目列表 |

### Current Detail

| 视角 | 明细内容 |
|:-----|:---------|
| Company | 全部项目列表 |
| Contract | 该合同下订单列表 |
| Project | 该项目下订单列表 |
| Order | 该订单下收入/成本流水 |

## 三、页面流

```
Business Analyzer（选择期间/视角/对象）
       ↓
KPI Summary 自动刷新
       ↓
Monthly Trend 自动刷新
       ↓
Alerts + Actions 自动刷新
       ↓
Detail 自动刷新
       ↓
点击 Alert → 跳转对应页面
点击 Action → 跳转对应页面
点击 Detail 行 → 跳转详情
```
