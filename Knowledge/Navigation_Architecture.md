## 二、各菜单详情

### Dashboard
| 属性 | 内容 |
|:-----|:------|
| 页面 | 经营总览 |
| 功能 | 收入/成本/利润/Gap 卡片 + 趋势图 + 预警列表 + 最近导入 |
| 入口 | 登录后默认 |
| 出口 | 点击任意卡片进入详情 |

### 合同中心
| 页面 | 功能 | 入口 | 出口 |
|:-----|:-----|:-----|:-----|
| 合同列表 | 搜索/筛选/查看合同 | Sidebar | → 合同详情 |
| 合同详情 | 合同信息 + 关联订单列表 | 点击列表 | → 订单列表 |

### 订单中心
| 页面 | 功能 | 入口 | 出口 |
|:-----|:-----|:-----|:-----|
| 订单列表 | 搜索/筛选 | Sidebar / 合同详情 | → 订单详情 |
| 订单详情 | 订单信息 + 收入/成本流水 | 点击列表 | → 收入列表 / 成本列表 |
| 订单管理 | 新建/编辑订单 | 工具栏按钮 | 返回列表 |

### 收入中心
| 页面 | 功能 | 入口 | 出口 |
|:-----|:-----|:-----|:-----|
| 收入流水 | 开票记录列表 | Sidebar / 订单详情 | → 收款 |
| 收款管理 | 收款记录 CRUD | 收入流水 | 返回 |
| 收入报表 | 收入汇总统计 | 侧栏 | — |

### 成本中心
| 页面 | 功能 | 入口 | 出口 |
|:-----|:-----|:-----|:-----|
| 成本流水 | 支出记录列表 | Sidebar / 订单详情 | → 付款 |
| 供应商管理 | 供应商/合同/单价 | Sidebar | 返回 |
| 付款管理 | 付款记录 CRUD | 成本流水 | 返回 |
| 成本报表 | 成本汇总统计 | 侧栏 | — |

### ERP 中心
| 页面 | 功能 | 入口 | 出口 |
|:-----|:-----|:-----|:-----|
| Import Workbench | 上传→解析→匹配→预览→确认 | Sidebar | → 导入结果 |
| Matching Workbench | 人工匹配界面 | Workbench | → Review |
| Review Workbench | 导入前审查 | Matching | → 确认 |
| Batch History | 历史批次列表 | Sidebar | → 详情 |

### AI 中心（预留）
| 页面 | 功能 | 入口 |
|:-----|:-----|:-----|
| AI 分析 | 经营数据 AI 分析 | Sidebar |
| AI 预警 | 自动预警列表 | Dashboard 预警卡片 |
| AI 建议 | AI 经营建议 | AI 分析页 |

### 系统管理
| 页面 | 功能 | 入口 |
|:-----|:-----|:-----|
| 数据字典 | 字典 CRUD | Sidebar |
| 操作日志 | 审计日志 | Sidebar |

## 三、导航规则

| 规则 | 说明 |
|:-----|:------|
| **Sidebar 为唯一主导航** | 所有一级/二级菜单通过 Sidebar 访问 |
| **Breadcrumb 显示路径** | 合同中心 > 合同列表 > 合同详情 |
| **详情页带返回** | 每个详情页顶部有 ← 返回按钮 |
| **新开页面不覆盖** | 详情页在同页打开（非新标签） |
| **导航保持状态** | 切换菜单后返回时保持搜索条件（可选） |
