# Revenue Business Field — 收入业务字段冻结

> **BDD-04A.1 P2 输出 · 业务冻结**
> 更新时间：2026-07-05
> 以下字段为经营字段，ERP 不覆盖。

---

## 一、invoice_stage（开票阶段）

| 属性 | 值 |
|------|-----|
| 字段 | `invoice_stage` |
| 类型 | String(50) |
| 必填 | ❌ 可选 |
| 来源 | M（人工） |
| ERP 覆盖 | ❌ |
| 默认值 | 无 |

### 标准值

| 值 | 说明 | 使用场景 |
|:----:|------|---------|
| 初验 | 项目初步验收阶段开票 | 工程施工类合同 |
| 终验 | 项目最终验收阶段开票 | 工程施工类合同 |
| 审计 | 审计后开票 | 政府/大型项目 |
| 月度 | 按月开票 | 长期服务类合同 |
| 一次性 | 一次性全额开票 | 小型项目/单项合同 |
| 其他 | 其他开票阶段 | — |

---

## 二、invoice_reason（开票原因）

| 属性 | 值 |
|------|-----|
| 字段 | `invoice_reason` |
| 类型 | String(50) |
| 必填 | ❌ 可选 |
| 来源 | M（人工） |
| ERP 覆盖 | ❌ |

### 标准值

| 值 | 说明 |
|:----:|------|
| 正常开票 | 常规开票流程 |
| 补开发票 | 补前期未开票部分 |
| 红冲 | 红字冲销 |
| 审减 | 审计审减后重新开票 |
| 设计变更 | 因设计变更导致的金额变化 |
| 其他 | 其他原因 |

---

## 三、冻结规则

| 规则 | 说明 |
|:----:|------|
| 两个字段均为**经营字段** | 非财务字段，ERP 不覆盖 |
| 来源为 **M（人工）** | 由业务人员录入 |
| 可选 | 不影响核心收入数据 |
| 标准值可扩展 | 后续可通过 SysDictionary 管理 |
