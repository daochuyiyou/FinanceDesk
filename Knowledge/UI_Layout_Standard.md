# UI Layout Standard — 统一布局标准

> **PDD-00 P3 输出 · 永久文档（Product SSoT）**
> 更新时间：2026-07-06
> **整个系统统一布局。禁止各模块自行设计。**

---

## 一、页面通用布局

```
┌──────────────────────────────────────────────────────────────┐
│ Header（48px）                                               │
│  Logo + 系统名  │  搜索  │  通知  │  用户  │  设置          │
├─────────┬────────────────────────────────────────────────────┤
│         │ Breadcrumb                                         │
│ Sidebar │ ─────────────────────────────────────────────      │
│ (220px) │ Toolbar（搜索/筛选/新建/导入/导出）                │
│         │ ─────────────────────────────────────────────      │
│ 收起 →  │ Summary Card Row（4 卡片）                          │
│  48px   │ ─────────────────────────────────────────────      │
│         │ Content Area（Table / Detail / Form）              │
│         │                                                    │
│         │                                                    │
│         │ Pagination                                         │
└─────────┴────────────────────────────────────────────────────┘
```

---

## 二、各区域规范

### Header（48px）

| 元素 | 位置 | 说明 |
|:-----|:-----|:------|
| Logo | 左上 | FinanceDesk 文字标 |
| 全局搜索 | 居中 | 预留 |
| 通知 | 右上 | 🔔 图标，未读红点 |
| 用户 | 右上 | 👤 头像 + 下拉菜单 |
| 设置 | 右上 | ⚙ 图标 |

### Sidebar（220px / 收起 48px）

| 规范 | 值 |
|:-----|:-----|
| 默认宽度 | 220px |
| 收起宽度 | 48px（仅图标） |
| 一级菜单 | 图标 + 文字 |
| 二级菜单 | 缩进 16px |
| 高亮 | 当前页面自动高亮 |
| 拖拽 | 支持排序（已实现） |

### Breadcrumb

| 规范 | 值 |
|:-----|:-----|
| 位置 | Header 下方 |
| 格式 | `首页 > 合同中心 > 合同列表` |
| 最后一项 | 当前页面（无链接） |
| 点击 | 回到任意上级 |

### Toolbar

| 元素 | 说明 |
|:-----|:------|
| 搜索输入框 | 关键字搜索 |
| 筛选器 | 下拉/日期范围 |
| 操作按钮 | 新建 / 导入 / 导出 |
| 刷新按钮 | 刷新当前数据 |

### Summary Card Row

| 规范 | 值 |
|:-----|:-----|
| 布局 | 一行 4 卡片 |
| 响应式 | 2 行 × 2 列（小屏） |
| 内容 | 数字 + 标签 + 趋势 |
| 点击 | 进入明细 |

### Content Area

| 类型 | 使用场景 |
|:-----|:---------|
| **Table** | 列表页（ProResizableTable） |
| **Detail** | 详情页（Descriptions） |
| **Form** | 新建/编辑（Ant Design Form） |
| **Workbench** | 导入工作台（Step + Card） |
| **Dashboard** | 驾驶舱（Statistic + Chart） |

---

## 三、页面类型

| 类型 | 布局 | 示例 |
|:-----|:-----|:------|
| **List Page** | Toolbar + Table + Pagination | 合同列表、订单列表 |
| **Detail Page** | Summary Card + Detail + Tab | 合同详情、订单详情 |
| **Form Page** | Form + Submit/Cancel | 新建合同、编辑订单 |
| **Workbench** | Steps + Cards + Confirm | ERP Import Workbench |
| **Dashboard** | Stat Cards + Chart + Alerts | 经营驾驶舱 |
| **Modal** | Centered Dialog | 确认对话框 |

---

## 四、统一组件

| 组件 | 使用规范 |
|:-----|:---------|
| **ProResizableTable** | 所有列表页必须使用 |
| **renderLongText** | 长文本列必须使用 |
| **DictSelect** | 字典字段必须使用 |
| **ErrorBoundary** | 每个页面需包裹 |
| **Statistic** | Summary Card 使用 |
| **Steps** | 工作流页面使用 |
